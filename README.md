# TrucksRegister

Olá!

O sistema foi criado e testado no Visual Studio 2022 Versão 17.4.2 Por favor utilize essa versão ou superior.

Testei também em outro computador com o Visual Studio 2019, executou a aplicação e os testes unitários sem problema.

O sistema foi criado utilizando o framework .NET CORE 3.1, por isso verifique se o seu Visual Studio 2022 ou 2019 tem essa versão do framework instalado.

O sistema foi testado com os ambientes de homologação (Debug) do Visual Studio 2022 e 2019: IIS Express e Kestrel. O navegador utilizado foi o Google Chrome.

As dependências já estão todas embarcadas, não é necessário instalar nenhum pacote extra. O banco de dados é criado no início da aplicação.

Testes Unitários: Execute os testes após executar a aplicação pela primeira vez. O projeto de testes segue junto e para rodar os testes basta clicar em testes, executar todos os teste.
